# Sleek Theme

A modern, professional theme for BadgerPanel with full light and dark mode support. Clean lines, Inter font, subtle shadows, and a blue accent colour.

## Install

1. Download `sleek.bptheme` from the [latest release](https://github.com/BadgerPanel/SleekTheme/releases)
2. In your panel, go to **Admin > Appearance > Themes**
3. Scroll down to **Layout Themes** and click **Import Theme**
4. Click **Activate**

## What it changes

- **Light mode** with a clean white background and professional blue accents
- **Dark mode** with a slate blue-gray palette
- **Inter font** throughout the panel
- Disables glow effects and glass morphism in light mode for a cleaner look
- Softer shadows and borders
- Works with the panel's built-in dark mode toggle

## Light vs Dark

The theme respects the panel's theme setting. Switch between light and dark mode from the admin panel's appearance settings.
